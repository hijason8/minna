# 單字表生成 — 標準提示詞（給網頁版 LLM）

請將下方整段複製到網頁版 LLM（如 ChatGPT、Claude、Gemini 等），  
產出的單字表可直接貼到本 App 的「解析單字表」或「匯入單字表」使用。

**從 API 取得同一份提示詞：** `GET /api/vocabulary/format` 回傳的 `standard_llm_prompt` 即為下方標準提示詞，可供介面一鍵複製。

---

## 標準提示詞（直接複製）

```
你是一位日文教材編輯。請根據我提供的課文、影片內容或主題，產出「單字表」。
產出時請嚴格遵守以下格式，不要加額外說明或 markdown 表格，只輸出純文字單字表。

【輸出格式】
- 每行一筆單字，共 3 或 4 欄，欄位之間用「Tab」分隔（不要用空格）。
- 第 1 欄：日文（漢字，若無漢字可留空或與讀音相同）。
- 第 2 欄：讀音（假名，必填）。
- 第 3 欄：中文意思（必填，可多個解釋用頓號或逗號分隔）。
- 第 4 欄（選填）：若為難點或需加強的單字，填 * ；否則可省略此欄。

第一行可選：若需要指定課程編號，請在第一行寫 lesson_id: 數字（例如 lesson_id: 3），然後空一行再開始單字。

【範例】
lesson_id: 3

受付	うけつけ	接待處、櫃檯
お手洗い	おてあらい	洗手間、廁所	*
自動販売機	じどうはんばいき	自動販賣機
いくら	いくら	多少錢
～階	～がい／かい	～樓層	*

【重要】
- 欄位之間只能有一個 Tab，不要用空格或多個 Tab。
- 不要輸出表頭列（如 kanji、kana、meaning）。
- 不要用 markdown、不要用程式碼區塊標記，只輸出上述格式的純文字。
```

---

## 簡短版（字數有限時用）

```
請產出日文單字表，每行一筆，欄位用 Tab 分隔：
第1欄=日文(漢字)、第2欄=讀音(假名)、第3欄=中文意思、第4欄選填*=難點。
第一行可寫 lesson_id: 3 指定課程。不要表頭、不要 markdown，只輸出純文字。範例：
受付	うけつけ	接待處、櫃檯
お手洗い	おてあらい	洗手間、廁所	*
```

---

## 本 App 可接受的分隔符

除 **Tab** 外，以下分隔符也會被自動辨識（但建議以 Tab 為主，LLM 較不易出錯）：

- Tab（`\t`）
- ` | `（前後有空格）
- `, `（逗號+空格）
- ` / `（前後有空格的斜線）

星標（第 4 欄）可接受：`*`、`★`、`⭐`、`1`、`true`、`是`、`星`、`難`、`y`、`yes`。
